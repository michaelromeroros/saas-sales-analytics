# saas-sales-analytics
End-to-end SaaS sales analytics project using BigQuery, SQL, and Tableau to analyze revenue trends, profitability, customer segments, and discount impact.

# SaaS Sales Analytics Dashboard
Project Overview

This project analyzes sales performance for a SaaS company using SQL and Tableau.

The objective was to identify revenue drivers, evaluate profitability, understand customer segmentation, and measure the impact of discounting strategies on business performance.

The analysis was conducted using Google BigQuery for data preparation and Tableau for dashboard development.

## Business Questions

- How is revenue evolving over time?
- Which industries generate the most revenue?
- Which customer segments contribute the most to sales?
- How do discounts affect profitability?
- What are the key business KPIs?

## Tech Stack

- Google BigQuery
- SQL
- Tableau
- GitHub

Primary analytical table:

sales_enriched

# Key fields:

- Sales
- Profit
- Customer ID
- Order ID
- Industry
- Segment
- Discount Bucket
- Product
- Order Date

All dashboard visualizations were built directly from this table to maintain a single source of truth.

# Dashboard Components

KPIs
- Total Revenue
- Total Profit
- Profit Margin %
- Total Customers

Visualizations
- Monthly Revenue Trend
- Revenue by Industry
- Revenue by Segment
- Discount Impact Analysis

# Key Insights

- Finance generated the highest revenue among all industries.
- SMB customers contributed the largest share of sales.
- Higher discount levels were associated with lower profit margins.
- Revenue showed an upward trend toward the end of the analysis period.

# Recommendations

- Prioritize high-performing industries such as Finance and Energy.
- Review discount policies for highly discounted transactions.
- Expand successful SMB acquisition strategies.
- Monitor profit margin alongside revenue growth.
